## Kullanılan Teknolojiler 🛠️
Proje geliştirilirken aşağıdaki teknolojiler kullanılmıştır:

- **Programlama Dili:** C# (.NET Framework) 
- **Veritabanı:** MySQL
- **Ara Yüz:** Windows Forms (WinForms), DevExpress
- **Mimari:** N Katmanlı Mimari
